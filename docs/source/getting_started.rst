=============
Getting started
=============


Installation
^^^^^^^^^^^^

Requirements
------------

* python 3 (>=3.7)
* numpy
* networkx
* pandas
* scipy
* scikit-learn
* statsmodels
* pydot
* torch>=1.7.1
* tick
* igraph
* lingam
* pgmpy<1.0.0
* tensorly=0.8.1
* tqdm>=4
* KDEpy>=1.1.12,<2



(For visualization)

* matplotlib
* graphviz
* pygraphviz (might not support the most recent Mac)


Install via PyPI
------------

To use CDMIR, we could install it using `pip <https://pypi.org/project/causal-learn/>`_:

.. code-block:: console

   (.venv) $ pip install CDMIR


Install from source
------------

For development version, please kindly refer to our `GitHub Repository <https://github.com/DMIRLAB-Group/CDMIR>`_.


Running examples
^^^^^^^^^^^^

For search methods in causal discovery, there are various running examples in the 'tests' directory in our `GitHub Repository <https://github.com/DMIRLAB-Group/CDMIR/cdmir>`_,
such as test_pc.py and test_desp.py.

For the implemented modules, such as independent test methods, we provide unit tests for the convenience of developing your own methods.
