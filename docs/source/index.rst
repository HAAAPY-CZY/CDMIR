Welcome to CDMIR's documentation!
=================================

**CDMIR** is a Python package for causal discovery, modeling, and reasoning,
developed by the DMIR Lab at Guangdong University of Technology (GDUT).
It provides a unified framework for building, evaluating, and interpreting
causal models, with user-friendly APIs that support both methodological
research and real-world applications.

.. note::

   CDMIR is under active development. For source code and usage examples,
   please refer to our `GitHub repository <https://github.com/DMIRLAB-Group/CDMIR>`_.
